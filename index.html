About the Course

Learn the essential skills needed for today’s digital workplace with our Microsoft Office beginner Course.

Key Learning Areas

  • Windows & File Management: Navigating the operating system, creating folders, organizing files
  • Microsoft Word: Create and format professional documents.
  • Microsoft Excel: Manage data, use formulas, and build simple spreadsheets.
  • Microsoft PowerPoint: Design impactful presentations with visuals and animations.
  • Microsoft Outlook: Organise emails, contacts, and calendars effectively.

What will I learn?

  • Create and format professional documents using Microsoft Word.
  • Use tables, images, and page layouts to enhance your Word projects.
  • Organise and manage data effectively in Microsoft Excel.
  • Apply basic formulas and functions to perform quick calculations.
  • Design impactful PowerPoint presentations with visuals and animations.
  • Use templates and slide layouts for professional presentation design.
  • Manage emails, contacts, and calendars in Microsoft Outlook.
  • Build confidence in completing everyday office tasks using Microsoft Office tools.
NOTE : Upon successful completion of the course, you will be presented with a Sher’s Study Centre certificate of completion. Sher’s Study Centre is not an accredited Training Centre, or Testing Centre, we simply provide you with the skills and confidence for today's digital workplace.
